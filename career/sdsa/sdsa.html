My career in US as a server administrator has begun!

My job is to provide stable web application server by monitoring the status of the servers using monitoring tool and analyzing log files to maintain the server’s stability as a server administrator.

Task Performed

Maintain and monitor the web application server, and develop using Java, Python, and any other languages for creating tools to maintain and monitor the servers.

• Monitor servers’ status to keep servers’ stability using Jenkins, Grafana, and Maxigent Plus.
• Keep WebLogic services updated by regular security patches from Oracle.
• Create Shell files using Python for making regular work process automatically.
• Keep JDK Version updated on Web Application Servers.
• Analyze log files to prevent problems that has high possibility that is going to make any errors or exceptions.
• Analyze log files to find a cause for errors or exceptions that occurred and plan for prevention.
• Using Docker to create an Environment for testing purpose.
